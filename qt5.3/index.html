Download and install Qt5.3.x from http://qt-project.org

Download and extract libusb-win32 from: http://sourceforge.net/projects/libusb-win32/

I downloaded: libusb-win32-bin-1.2.6.0.zip (http://sourceforge.net/projects/libusb-win32/files/libusb-win32-releases/1.2.6.0/libusb-win32-bin-1.2.6.0.zip/download)

Extract into a directory. In the include directory, copy lusb0_usb.h to usb.h

Download and install USBXpress Development Kit for: http://www.silabs.com/products/mcu/Pages/USBXpress.aspx

I downloaded the 3.5.1 version. (http://www.silabs.com/Support%20Documents/Software/USBXpress_Install.exe)

Extract into a directory. You do not need to install the drivers.

Download and extract D2XX drivers from: http://www.ftdichip.com/Drivers/D2XX.htm

I downloaded: CDM v2.10.00 (http://www.ftdichip.com/Drivers/CDM/CDM%20v2.10.00%20WHQL%20Certified.zip)

You will need flex. Get a copy from: http://gnuwin32.sourceforge.net/packages/flex.htm

I used the Binaries zip. (http://gnuwin32.sourceforge.net/downlinks/flex-bin-zip.php)

Open the zip file, go into the bin folder and extract all files into the Qt bin folder.

You will need bison. Get a copy from: http://gnuwin32.sourceforge.net/packages/bison.htm

I used the Binaries zip. (http://downloads.sourceforge.net/gnuwin32/bison-2.4.1-bin.zip)

Open the zip file, go into the bin folder and extract all files into the Qt bin folder.

Get a copy of the GC master code. (https://github.com/GoldenCheetah/GoldenCheetah/archive/master.zip)

Extract the files.

In qwt folder, copy qwtconfig.pri.in to qwtconfig.pri
In src folder, copy gcconfig.pri.in to gcconfig.pri
